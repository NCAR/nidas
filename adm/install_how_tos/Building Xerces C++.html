Installing Xerces for ADS3

These instructions are current as of xercesc-c version 2.6.  They also assume you're running sh/bash. If you're using csh/tcsh, substitute "setenv" for "export", as appropriate.

First, fetch the source code for Xerces C++ from http://xml.apache.org/xerces-c/.

Cross-compiling for ARM

cd /tmp
tar xzf xerces-c-current.tar.gz
export XERCESCROOT=$PWD/xerces-c-src_2_6_0
cd $XERCESCROOT/src/xercesc

Edit runConfigure so that it accepts arm-linux-g++.   There is a case statement at line 631.  Add arm-linux-g++ as an
acceptible compiler in the group that contains g++:

case $cppcompiler in
  ...
   xlC* | xlc* | g++ | c++ | cc | CC | aCC | icc | ICC | cxx | ecc | QCC | arm-linux-g++)
      CXX="$cppcompiler"
  ...

Make sure the arm-linux-g++ is found in your path:

which arm-linux-g++

Then build it:

export CXXFLAGS="-O2"
export CFLAGS="-O2"
./runConfigure -p linux -c arm-linux-gcc -x arm-linux-g++ -C --host=arm-linux -C --prefix=/net/opt_lnx/ads3/arm
make
make install
make distclean

Building for X86

./runConfigure -p linux -c gcc -x g++ -C --prefix=/net/opt_lnx/ads3/x86
make
make install
make distclean
