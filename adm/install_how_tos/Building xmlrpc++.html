Building xmlrpc++ for ADS3

Download the library's source from the web

These instructions are current as of xmlrpc++ version 0.7.
Obtain the source code for xmlrpc++ from http://xmlrpcpp.sourceforge.net

ssh orion
cd /tmp/
tar -xzvf xmlrpc++0.7.tar.gz
cd xmlrpc++0.7

Remove the following unsupported compiler warning:  -Wstrict-prototypes

Apply the following patch by executing this command:  patch -p1 < Validator.cpp.patch

 Set up the include paths for both libraries:
mkdir /net/opt_lnx/ads3/arm/include/xmlrpc++
mkdir /net/opt_lnx/ads3/x86/include/xmlrpc++

Building for ARM

Edit the Makefile to use the following compiler:   CXX = arm-linux-g++
 Build and install:
make clean
make
cp -p src/*.h /net/opt_lnx/ads3/arm/include/xmlrpc++/
cp -p libXmlRpc.a /net/opt_lnx/ads3/arm/lib/

Building for x86

Edit the Makefile to use the following compiler:   CXX = g++
 Build and install:
make clean
make
cp -p src/*.h /net/opt_lnx/ads3/x86/include/xmlrpc++/
cp -p libXmlRpc.a /net/opt_lnx/ads3/x86/lib/
